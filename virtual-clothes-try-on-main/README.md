## 🧠 What is it?

<table style="border: 1px solid #ccc; border-radius: 8px; padding: 15px; background-color: #f5faff; width: 100%;">
  <tr>
    <td>
      <strong>Virtual Clothes Try-On</strong> is an <strong>AI-based application</strong> that lets users try on shirts virtually using a webcam.
      Built with <strong>Python</strong> and <strong>OpenCV</strong>, it uses real-time image processing to overlay clothes on the user’s body.
      This enhances the online shopping experience through interactive visualization and helps users make better purchasing decisions without needing to try clothes physically.
    </td>
  </tr>
</table>
